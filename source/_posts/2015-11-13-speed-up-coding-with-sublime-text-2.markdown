<strong>switch between groups: </strong>
```
control + GROUP_NUMBER
```

<strong>switch to sidebar</strong>
```
control + 0
```

<strong>switch between tabs:</strong> 
```
option + TAB_NUMBER
```

<strong>search and open file</strong>
```
option + p
```
